---
description: Diagnose what kind of stuck you are and produce one targeted intervention plus a fallback.
argument-hint: [describe what's happening — short is fine]
---

The user is stalled mid-task or at the start of one. Diagnose the *type* of stuck and intervene narrowly.

**Process:**
1. Read what the user said. If they said almost nothing, ask **one** small question: *"What's happening right now — open the task and froze, or never opened it, or got distracted off it?"*
2. Classify. Pick ONE of:
   - **Unclear** — they don't actually know what the task is.
   - **Too big** — the task is really a project.
   - **Boring** — low stimulation, no friction-reduction will fix it without a stimulation hack.
   - **Scary** — fear of the outcome, judgment, or finishing.
   - **Emotionally loaded** — touches a relationship, conflict, identity.
   - **Perfectionism** — done feels unsafe.
   - **Missing dependency** — blocked on info, decision, or another person.
   - **Low energy** — depletion is real; pushing harder will fail.
   - **Context switching** — head is still on the previous task.
3. Apply the targeted intervention from the table below.
4. Always provide a **fallback easier version** and a **restart phrase**.

**Interventions:**

| Stuck type | Intervention | Fallback |
|---|---|---|
| Unclear | Spend 5 minutes writing what done looks like, in one sentence. Don't start the work. | "Write the goal sentence. That's the whole task today." |
| Too big | Stop. Run /plan-project on this. Then do the first 10-minute action only. | "Define one milestone. Stop." |
| Boring | Pair with a wanted thing — coffee, podcast, a walk after. Stand at the desk. Set a 15-min timer. | "Do the first 5 minutes." |
| Scary | Name the specific feared outcome in one sentence, then run /reframe. | "Write the first ugly sentence. That's it." |
| Emotionally loaded | Switch modes — go for a 10-minute walk, then return and write a draft you won't send. | "Draft. Don't send. Come back tomorrow." |
| Perfectionism | Set a deliberately-bad-first-draft rule. Define a stop time (not a quality bar). | "Time-boxed bad version. Ship to nobody." |
| Missing dependency | Stop the task. Send the unblocking message right now. Move on. | "Send the email. Pick up something else." |
| Low energy | Recovery, not effort. Water, food, fresh air, 20 minutes. Then reassess. | "Today is a recovery day. Pick the smallest must." |
| Context switching | 5-minute close-out of the previous task: where it stopped, next step. Then start. | "Write the 'next time' note for previous task." |

**Output shape:**

```
DIAGNOSIS
- <stuck type>: one sentence why.

INTERVENTION
- <the targeted move, in 1–3 lines>

NEXT ACTION
- <one specific thing to do in the next few minutes>

FALLBACK (if even that is too much)
- <smaller version>

RESTART PHRASE
- A short sentence to say to yourself when you fall off again. Example:
  "I get to start over. The work doesn't care."
```

Do not list all nine interventions. Pick one. Be specific. Be brief.
